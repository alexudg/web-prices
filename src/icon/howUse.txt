* copy folder 'webfonts' to folder assets
* create folder 'css' aside of folder 'webfonts'
* copy from 'css/all.css' to folder 'css' aside of folder 'webfonts'
* insert link in html file...
  <link rel="stylesheet" href="assets/css/all.css"> 

* use icon example...

  <i class="fab fa-facebook fa-2x"></i> 

  ...where fa-2x is size
     xs = xtra small .75em
     sm = small .875em
     lg = large 2.33em
     2x = 2em
     ... 
     10x = 10em

* custom size and color ...
  <span style="font-size: 18px; color: red;">
    <i class="fab fa-facebook fa-2x"></i>
  </span>
